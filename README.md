### музыкальный плеер под терминал
### необходимо наличие sounddevice, sounfile и numpy (питоновские библиотеки)
### что бы плеер видел музыку, нужно указать в самом коде путь до родительской папки, учитывая придирки операционной системы
```bash
directory = "/path/to/your/papka"
```
### плеер может работать с разными размерами терминала т.к. почти все переменные, связаные с расположением текстовых приколов, привязаны к константам ширины и высоты терминала
### учитывая что это КОНСТАНТЫ - размер терминала нельзя изменять во время работы программы
### из рабочих клавишь только стрелочки вверх[k], вниз[j], влево(для промотки на предыдущую песню)[h], вправо(для перемотки на некст песню)[l], enter(для перехода в плейлист или включения песни), esc(для выхода из плейлиста или для выкдючения программы), space(пауза)
### песни играются друг за другом и их нелья перематывать(в плане времени)
